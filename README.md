# Obsidian Tokenz

Use any kind of short codes in Obsidian!

Comes with a pre-defined map not only for
- emojis (:smile: 🙂, :wink: 😉, ...), but also for the classic, good old
- **smileys** ( **:-)** 🙂, **8-D** 😎, ...).
